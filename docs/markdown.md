# 区块元素
## 链接
  [链接1](https:www.xiaor.com "title") 跳转到网页

  [链接2](README) 跳转到文件

  参考式的链接是在链接文字的括号后面再接上另一个方括号，而在第二个方括号里面要填入用以辨识链接的标记

  [参考式][foo]

  [foo]: https:www.xiaor.com

  **自动链接**

  <https:www.xiaor.com>

  **隐式链接(省略链接id标记，相当于简写)**

  [小r][]是一个很可爱的[姑娘][]啊

  [小r]: https:www.xiaor.com
  [姑娘]: https:www.xiaor.com
## 强调
星号和_都可以作为强调的符号

*星号强调 会变成[em]()标签 一个星号*

_下划线强调 会变成[em]()标签 一个底线号_

**星号强调 会变成[strong]()标签 2个星号**

__下划线强调 会变成[strong]()标签 2个底线号__
